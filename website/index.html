bibtex.js

This JavaScript version of BibTeX was created by converting WEB to Pascal (via tangle), Pascal to C (via web2c), C to LLVM bytecode (via LLVM), and LLVM bytecode to Javascript (via emscripten). It was inspired by texlive.js, a conversion of pdftex to JavaScript. With these two programs now running in JavaScript, most of the professional typesetting done in academia and anywhere that uses LaTeX can now be done completely from within the browser, in principle. In practice, we have more work to do, namely, systems for 1) automatic dependency resolution and 2) a JavaScript implementation of a latex/bibtex Makefile such as LaTeX-Mk.

From the BibTeX man page: BibTeX reads the top-level auxiliary (.aux) file that was output during the running of latex or tex and creates a bibliography (.bbl) file that will be incorporated into the document on subsequent runs of LaTeX or TeX.

BibTeX looks up, in bibliographic database (.bib) files specified by the \bibliography command, the entries specified by the \cite and \nocite commands in the LaTeX or TeX source file. It formats the information from those entries according to instructions in a bibliography style (.bst) file (specified by the \bibliographystyle command), and it outputs the results to the .bbl file.
